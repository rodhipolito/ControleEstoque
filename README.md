# ControleEstoque
Controle de Estoque


Software controle de estoque:

Tela Inicial 
Login:[Usuario]
Senha:[definida pelo usuario]

alertas em caso de erro de login

Após entrar no login ele mostra 

Dashboard:
Visão geral do status do estoque, incluindo níveis de produtos, pedidos pendentes e alertas.

Módulo de Entrada e Saída:
Para registrar a entrada e saída de produtos.


Notificações:
Sistema de alertas para reposição de estoque quando faltar 2 unidades.

Histórico de Movimentações:
Registro de todas as movimentações de estoque para auditoria e controle.
